Теорія чисел. Подільності та остачі.

Деякі цікаві задачі для школярів на тему "Теорія чисел. Подільності та остачі."

Означення. Цілі числа $a$ та $b$ називаються еквівалентними по модулю $m$, якщо їхня різниця ділиться на $m$ без остачі. Позначаємо $a\equiv b (\mod m)$.

Для операції порівняння по модулю виконуються наступні властивості:

  • $a\equiv b (\mod m)$ => $a+c\equiv b+c (\mod m)$ для довільного $c$.
  • $a\equiv b (\mod m)$ => $a^{k}\equiv b^{k} (\mod m)$ для довільного $k\in \mathbb{N}$.
  • $a\equiv b (\mod m)$ => $ac\equiv bc (\mod mc)$ для довільного $c$.
  • $ad\equiv bd (\mod m)$ => $ad\equiv bd (\mod m)$ якщо НСД($m$,$d$)=1.
  • $a\equiv b (\mod m)$ => $a\equiv b+cm (\mod m)$ для довільного- цілого $c$.

Китайська теорема про остачі. Нехай $a_{1},...,a_{n}$ - попарно прості натуральні числа, то для довільних натуральних $r_{1},...,r_{n}$, таких, що $0\leq r_{i}< a_{i}$ для кожного $i$, $0\leq i\leq n$, існує єдине число $x$, таке, що $x\equiv r_{i}(\mod a_{i})$.


Задача 1. Знайдіть останню цифру числа $1^{2}+2^{2}+...+99^{2}$.


Задача 2. Знайдіть остачу при діленні числа $6^{2025}$ на $7$.


Задача 3. Чи може сума квадратів двох непарних чисел бути квадратом цілого числа?


Задача 4. Є 100 купюр двох номіналів, $a$ та $b$ монгольських тугриків, $a\neq b (\mod 101)$. Доведіть, що можна вибрати декілька купюр так, що отримана сума (в тугриках) ділиться на 101.


Задача 5. Чи можна розставити по кругу числа $1,2,...,60$ таким чином, щоб сумма кожних двох чисел, між якими знаходяться шість чисел, ділилася на 7.


Задача 6 (обласна олімпіада 2026). Знайдіть усі пари натуральних чисел $(a,b)$, що задовольняють умови: число $b + 1$ ділиться націло на число $a$ та число $2a + 1$ ділиться націло на число $3b$.


Задача 7 (обласна олімпіада 2026). Знайдіть всі четвірки чисел $(n, p, q, r)$, де $n$ - натуральне число, а $p$, $q$, $r$ - прості числа, такі що $p^{2} + 2qr = 6^{n} + 1$.


Задача 8. Знайти всі натуральні числа, які дають остачі 1,2,3,4 по модулям 5,7,11,12.
