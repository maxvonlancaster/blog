Комплексні числа

Деякі цікаві задачі для школярів на тему "Комплексні числа"

Комплексні числа: розширення множини дійсних чисел, числа вигляду $x+iy$, де $i$ - уявна одиниця ($i^{2}=-1$), а $x$ та $y$ - дійсні числа.

Візуалізуємо комплексні числа на площині!




Комплексні числа також можна записувати у вигляді $x = r(\cos \alpha + i \sin \alpha)$, де $r$ - дійсне число, $\alpha$ кут між $0$ та $2\pi$


Формула Муавра: для $x=r(\cos \varphi + i\sin \varphi)$ справджується: $$x^{n}=r^{n}(\cos n\varphi + i\sin n\varphi)$$

Доведення методом математичної індукції.

База індукції - при $n = 1$: $(r(\cos x + i\sin x))^{1} = r(\cos 1x + i\sin 1x$)
Припустимо тепер, що для натурального $k$ справджується $$(r(\cos x + i\sin x))^{k} = r^{k}(\cos kx + i\sin kx)$$ Розглянемо тепер випадок $k+1$: $$(r(\cos x + i\sin x))^{k + 1} = r^{k+1}(\cos kx + i\sin kx)(\cos x + i\sin x) =$$ $$=r^{k+1}(\cos kx \cos x -\sin kx \sin x + i(\sin kx \cos x + \sin x \cos kx) )=$$ $$= r^{k+1}(\cos(kx + x) + i\sin(kx + x)) = r^{k+1}(\cos(x(k+1)) + i\sin(x(k+1)))$$

Задача 1. Довести, що для довільних двох чисел, кожне з яких є сумою двох квадратів, їхній добуток також є сумою двох квадратів.

Нехай $a = x^{2} + y^{2}$, $b = u^{2} + v^{2}$
Тоді $a = x^{2} - i^{2} y^{2} = (x-iy)(x+iy)$, аналогічно $b = (u-iv)(u+iv)$.
$ab = ((x-iy)(u+iv))((x+iy)(u-iv)) =$ $ = (xu + yv - i(yu - xv))(xu + yv + i(yu - xv)) = $
$= (m-in)(m+in) = m^{2} + n^{2}$.


Задача 2. При яких $p$ і $q$ многочлен $x^{4}+1$ ділиться на $x^{2}+px+q$

Маємо, що $x^{4} = - 1 = \cos \pi + i \sin \pi$:
За формулою Муавра, тоді $x = \cos \frac{\pi + 2\pi k}{4} + i \sin \frac{\pi + 2\pi k}{4} = \pm\frac{\sqrt{2}}{2} \pm i \frac{\sqrt{2}}{2}$.
А отже, такими ж мають бути і розв'зки рівняння $x^{2}+px+q = 0$
За формулою Вієта, $x_{1}x_{2} = q$, $- x_{1} - x_{2} = p$, де $x_{1}, x_{2} \in \{\pm\frac{\sqrt{2}}{2} \pm i \frac{\sqrt{2}}{2}\}$


Задача 3. Нехай $\sin \alpha = \frac{3}{5}$. Доведіть, що $\sin 25\alpha$ має вигляд $\frac{n}{5^{25}}$, де $n$ - ціле число, що не ділиться на 5.

Нехай $\sin \alpha = \frac{3}{5}$, нам потрібно довести, що $\sin 25 \alpha = \frac{n}{5^{25}}$.
Покладемо $x = \frac{4}{5} + i\frac{3}{5}$, тоді $$x^{25} = (\frac{4}{5} + i\frac{3}{5})^{25} = $$
$$= (\frac{4}{5})^{25} + 25 (\frac{4}{5})^{24}\frac{3}{5} + ... + (i\frac{3}{5})^{25} = $$
$$= A + i\frac{3^{25} + 5m}{5^{25}} = \cos 25\alpha + i\sin 25 \alpha,$$ де $A$, $m$ - дійсні числа


Задача 4. Доведіть, що якщо $x+iy=(s+it)^{n}$, то $x^{2}+y^{2}=(s^{2}+t^{2})^{n}$

$(s + it)^{n} = (r(\cos \alpha + i \sin \alpha))^{n} = r^{n} (\cos n\alpha +i\sin n \alpha ) = x + iy$
$x^{2} + y^{2} = r^{2n}\cos^{2} n\alpha + r^{2n}\sin^{2} n\alpha = r^{2n}(\cos^{2} n\alpha + \sin^{2} n\alpha) = $
$= r^{2n} = r^{2n}(\cos^{2}\alpha + \sin^{2}\alpha )^{n} = (r^{2}\cos^{2}\alpha + r^{2}\sin^{2}\alpha )^{n} = (s^{2} + t^{2})^{n}$
